## 打字通（单页版）

一个基于 HTML，CSS,JavaScript 和 Vue 3 的打字练习页面，支持中文/英文词库、自定义文本、计时测速、更换背景、账号登录与历史成绩、主题和背景音效等。

## 功能特性

- 中文/英文词库切换，行间滚动保持居中。
- 自定义文本：可添加、选择并再次编辑。
- 账号与历史：登录后记录并查看个人成绩；未登录查看历史时会提示先登录；游客不记录成绩。
- 主题与背景：多主题色、背景图，自定义背景上传，背景音乐播放/上传，音量控制。
- 键盘反馈：虚拟键盘指法高亮，按键音效。
- 统计与提示：计时、WPM/字速、完成提示弹窗“您已完成本次打字，可在历史成绩中查看本次成绩”。进度条同步展示完成比例，可暂停/结束并重置

## 运行步骤

1. 获取代码：将项目文件解压或克隆到本地，保证 [index.html](index.html) 与资源文件在同一目录。
2. 准备资源：如需自定义背景/BGM，将图片、音频文件放在项目根目录（可选，未放置则使用默认渐变和静音）。
3. 启动方式（二选一）：
   - 直接打开：双击 [index.html](index.html) 用 Chrome/Edge 打开。
   - 本地服务：在项目目录打开终端，运行 `python -m http.server 8000` 或 `npx serve .`，然后在浏览器访问 http://localhost:8000。
4. 允许音频：首次播放 BGM/键音时，若被浏览器阻止，请先点击页面任意位置再播放。
5. 登录体验（可选）：点击右上角“登录 / 注册”创建本地账号，以便保存历史成绩。

## 使用指南

- 选择文本：顶部语言按钮切换后，点击列表中的卡片开始练习；或点击“自定义文本”卡片编辑并应用自定义内容。
- 输入练习：聚焦隐形输入框后直接键入（中文逐字匹配，英文整行匹配）。当前字符会高亮，错误字符在英文模式下标红。
- 控制面板：
  - 计时区域：暂停/继续、结束记录、查看历史。
  - 主题/背景：选择主题点、背景缩略图或上传自定义背景，需还原时点“取消背景”。
  - 音效：键音开关；BGM 播放/暂停、音量滑杆、上传音频、恢复默认。
  - 账号：点击“登录 / 注册”完成本地账户创建或登录；退出后历史清空。

## 本地存储约定

- 用户信息：`typer_users`（用户名: 密码）、`typer_current_user`（当前用户名）。
- 历史成绩：`typer_history`，按用户名分组存储每次 WPM/CPM、用时、文本与时间戳。

## 依赖与技术栈

- CDN：Vue 3（Composition API）、Tailwind CSS。
- 纯前端实现，无构建步骤，无额外依赖。

## 目录结构

- [index.html](index.html)：页面、样式与逻辑的主体文件。
- [readme.md](readme.md)：项目说明（本文件）。
- 其他静态资源：`bgm.mp3`、`background-*.png` 等（可选）。

## 常见问题

- BGM 无法播放：首次操作需用户交互；确认 bgm.mp3 存在或上传自定义音频。
- 背景不生效：检查背景图片是否与 HTML 同级；可点击“取消背景”恢复默认渐变。
- 历史为空：需先登录；退出登录会清空当前用户的历史列表。
